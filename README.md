1. **Fat Content Breakdown**

4. **Outlet Establishment Trends**

   - A major dip is observed in outlets opened in **2011**.

   - Medium: 42.3%  
   - Small: 37%  
   - Large: 20.7%

6. **Category Performance by Fat Content**

   - **Regular**: Fruits and Vegetables performed best.
